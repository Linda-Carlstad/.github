# Linda Carlstad

Det officiella Git-förrådet för studentföreningen Linda Carlstad i Sverige, Karlstad. Föreningen står under och med stöd från Karlstads universitet. Linda Carlstad är en förening för studenter som studerar datavetenskap, webbutveckling och mjukvarusystemdesign.
